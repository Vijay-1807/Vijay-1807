## ◈ Featured Projects

<details>
<summary><strong>⬡ &nbsp;AI-Powered Crop Disease Detection System</strong></summary>
<br/>

An end-to-end machine learning web application that classifies crop leaf diseases from image input using a trained convolutional neural network, served via a Flask REST API and a responsive front-end interface. Designed to assist farmers with rapid, on-device diagnosis.

| Attribute | Details |
|-----------|---------|
| **Stack** | Python · Flask · TensorFlow/Keras · React · MongoDB |
| **Scale** | Multi-class classification across 38 disease categories |
| **Performance** | ~94% validation accuracy on PlantVillage dataset |
| **Security** | Input validation, rate limiting, JWT-protected endpoints |
| **Impact** | Reduces field diagnosis time from hours to seconds |
| **Repository** | [github.com/Vijay-1807](https://github.com/Vijay-1807) |

The system ingests raw leaf images, preprocesses them through a normalisation pipeline, and routes them to a fine-tuned MobileNetV2 model. Results are returned with confidence scores and actionable treatment recommendations. The architecture is stateless and horizontally scalable.

</details>

<details>

A production-ready e-commerce application built on the MERN stack, featuring a complete purchase lifecycle — product catalogue, cart management, order tracking, and payment gateway integration. Architected with a RESTful API backend and a React single-page application front-end.

| Attribute | Details |
|-----------|---------|
| **Stack** | MongoDB · Express.js · React · Node.js · Bootstrap |
| **Scale** | Multi-user concurrent sessions with persistent cart state |
| **Performance** | Sub-200ms API response times with MongoDB indexing |
| **Security** | bcrypt password hashing · JWT auth · CORS policies |
| **Impact** | End-to-end retail workflow from discovery to fulfilment |
| **Repository** | [github.com/Vijay-1807](https://github.com/Vijay-1807) |

Implemented role-based access control distinguishing admin and customer flows. Admin dashboard supports real-time inventory management and order status updates. Front-end state is managed via React Context API with optimistic UI updates for cart operations.

</details>

<summary><strong>⬡ &nbsp;Intelligent Student Performance Analytics Dashboard</strong></summary>
<br/>

A data analytics platform that ingests academic records and produces actionable performance insights through interactive visualisations and a machine learning model that predicts at-risk students based on historical patterns.

| Attribute | Details |
|-----------|---------|
| **Stack** | Python · Pandas · Scikit-learn · Flask · Matplotlib · MySQL |
| **Scale** | Batch processing for cohort-level academic datasets |
| **Performance** | Logistic regression model with 88% prediction accuracy |
| **Security** | Role-gated dashboards for faculty vs administrator access |
| **Impact** | Enables early intervention for academically at-risk students |
| **Repository** | [github.com/Vijay-1807](https://github.com/Vijay-1807) |

The ETL pipeline cleans and transforms raw CSV exports from the institution's LMS. Feature engineering extracts attendance rates, assignment submission velocity, and grade trends as model inputs. Visualisation layer built with Matplotlib and served via a Flask template engine.

</details>
